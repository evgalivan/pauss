Программа автоматизации управления для снятия переодических спектров газов и газовых смесей (ПАУСС)

Целью работы программы является управление программой GASpec по интерфейсу: Control.ini, Status.ini.

Установка программы:
 Для работы программы требуется:
	старше python v 3.7
	установленное ПО FSpec с модулем GASpec
	ВАЖНО! Необходимо убрать требование "запускать с правами администратора" с программы FSpec
	
 Скопируйте содержимое архива с программой в удобную для вас папку на компьютере.

Запуск программы и использование:
	Программа запускается:
	a) через python
		Откройте pauss.py через python.exe (по пути установки python обычно (C:\Users\User\AppData\Local\Programs\Python))
	б) через cmd
	из текущей папки, где находится программа pauss.py введите команду одну из следующих команд:
		py pauss.py
		python pauss.py
		python3 pauss.py

Дальше, действуйте по инструкциям выводимым программой.
